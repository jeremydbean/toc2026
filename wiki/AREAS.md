
<p><font size="3">This is the easiest section to do. The format
is as follows:</font></p>

<p><font size="3">#AREA &lt;area-name:string&gt;</font></p>

<p><font size="3">The 'area-name' can be any string. The 'areas'
command provides a list of areas, so it's worth while to follow
the standard Merc format for this string. This format would look
as follows, to take an example from an area already designed:</font></p>

<table border="2">
    <tr>
        <td><font size="3">#AREA </font></td>
        <td><font size="3">{30 60}</font></td>
        <td><font size="3">Soul </font></td>
        <td><font size="3">Daggoth's Thrahl</font></td>
        <td>&nbsp;</td>
    </tr>
    <tr>
        <td><font size="3">1</font></td>
        <td><font size="3">8</font></td>
        <td><font size="3">16</font></td>
        <td><font size="3">24</font></td>
        <td><font size="3">The numbers denote the character space
        position to allow the area</font></td>
    </tr>
    <tr>
        <td>&nbsp;</td>
        <td>&nbsp;</td>
        <td>&nbsp;</td>
        <td>&nbsp;</td>
        <td><font size="3">list file to line up nicely :), Please
        Adhere to them.</font></td>
    </tr>
</table>

<p><font size="3">The first two numbers are the recommended level
range of the area. The first name is your own name or alias as it
shows with the 'areas' command, and the last phrase is the name
of the area itself.</font></p>
